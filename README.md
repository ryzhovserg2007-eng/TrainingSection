# TrainingSection - Приложение для управления группами тренировок

## 📊 Описание проекта

Приложение для сбора статистики эффективности работы тренеров. Позволяет управлять тренерами, спортсменами, группами тренировок и отслеживать рейтинг тренеров.

## 🎯 Функциональность

### Базовая часть (5 баллов)
- Управление тренерами и спортсменами
- Создание групп тренировок (1 тренер + массив спортсменов)
- Автоматическое создание 3+ тренеров и 5+ групп при первом запуске
- Выпадающий список групп
- Таблица участников группы с возможностью добавления новых
- Сохранение/загрузка данных в JSON

### Дополнительная часть (3 балла)
- Библиотека классов "Model" с бизнес-логикой
- Сортировка группы по ФИО
- Фильтрация по полу и возрасту
- Таблица тренеров с количеством групп и статистикой (мужчины/женщины, средний возраст, медиана)

### Продвинутая часть (2 балла)
- XML сериализация (дополнительно к JSON)
- Система рейтинга тренеров
- Фидбек от спортсменов с оценкой
- Интерфейс IReportable для генерации отчетов
- Выбор формата отчета (JSON/XML)

## 🖥️ Структура интерфейса

### Вкладка 1: Управление группами
- **Выпадающий список групп** - выбор группы для работы
- **Таблица участников** - показывает всех участников выбранной группы
  - Столбцы: ФИО, Пол, Возраст
  - Автоматически обновляется при выборе другой группы
- **Поля для добавления** - текстовые поля для ввода ФИО, пола, возраста
- **Кнопка "Добавить участника"** - добавляет нового спортсмена в группу
- **Кнопка "Удалить"** - удаляет выбранного участника
- **Кнопки фильтрации** - фильтр по полу (М/Ж) и возрасту

### Вкладка 2: Статистика по тренерам
- **Таблица тренеров** - показывает всех тренеров
  - Столбцы: ФИО тренера, Рейтинг, Количество групп, % Мужчин, % Женщин, Средний возраст, Медиана возраста
  - Таблица не редактируемая (只读)

### Вкладка 3: Фидбек
- **Выпадающий список спортсменов** - выбор спортсмена
- **Выпадающий список групп** - выбор группы (его группа)
- **Поле для оценки** - оценка тренера (1-10)
- **Кнопка "Оставить фидбек"** - активна только ��сли выбраны спортсмен и группа
- **Таблица фидбеков** - история всех оценок

## 🛠️ Как использовать

1. **При запуске** приложение автоматически создаст:
   - 3+ тренеров
   - 5+ групп с разными тренерами
   - Тестовых спортсменов

2. **Добавление участника:**
   - Выберите группу из выпадающего списка
   - Введите ФИО, пол (М/Ж), возраст
   - Нажмите "Добавить участника"
   - Данные сохранятся в JSON

3. **Просмотр статистики:**
   - Перейдите на вкладку "Статистика по тренерам"
   - Увидите всех тренеров с их рейтингом и группами

4. **Оставление фидбека:**
   - Перейдите на вкладку "Фидбек"
   - Выберите спортсмена и его группу
   - Введите оценку (1-10)
   - Нажмите "Оставить фидбек"
   - Спортсмен сможет оставить оценку только один раз для каждой группы

5. **Выбор формата отчета:**
   - Используйте выпадающий список "Формат отчета" (JSON/XML)
   - При смене формата данные автоматически конвертируются
